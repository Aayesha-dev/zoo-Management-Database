# 🦁 Zoo Management System - Database

A comprehensive **Zoo Management System** database built with **Microsoft Access**.  
This database manages **animals, habitats, staff, visitors, financial records, security, food supply, and more**.

---

## 🎯 Database Features

| Feature | Description |
|---------|-------------|
| **🐘 Animal Management** | Track species, health records, habitat, and feeding schedules |
| **🏠 Habitat Management** | Manage enclosures with capacity and environmental conditions |
| **👨‍🔧 Staff Management** | Track zookeepers, security staff, and their assignments |
| **🍽️ Food Supply** | Manage food inventory and supplier details |
| **💰 Financial Records** | Track revenue, expenses, and budget |
| **🎟️ Visitor Management** | Record visitor entries, ticket sales, and cafe/play area visits |
| **🚗 Parking & Transport** | Manage parking lots and transport vehicles |
| **🏥 Medical Records** | Track animal health and treatment history |
| **🆘 Emergency Plans** | Store emergency procedures and protocols |
| **🔐 Security** | Manage security staff and safety protocols |
